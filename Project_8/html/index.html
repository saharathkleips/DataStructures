The specifications of this project match those of the book C++ Data Structures - A Laboratory Course (3rd Edition) Project 8. Mathematical expressions form a hierarchy of operations built upon the addition, subtraction, multiplication, and division operators. This hieracrhy of operations to preform on an expression can be explicitly expressed using a binary tree, specifically called a expression tree.
Notes:
Infix notation is when each operator is placed between its operands: ( 1 + 3 ) * ( 6 - 4 )
Prefix notation is when each operator is placed immediately before its operands: * + 1 3 - 6 4
